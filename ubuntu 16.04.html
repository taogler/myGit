ubuntu 16.04

ubuntu

一. 安装ubuntu 16.04

  1. 打开网页http://releases.ubuntu.com/16.04/
  2. 下载ubuntu-16.04-desktop-amd64.iso
  3. 在windows下利用Universal-USB-Installer或在ubuntu下利用Startup Disk Creator制作启动盘.
  4. 利用U盘启动进行安装, 按指示安装, 选择不连接网络安装, 选择手动分区.

二. 连接网络

  • 连接无线网,
  • 采用mentohust联接有线网
    1. 下载mentohust_0.3.4-1_amd64.deb
    2. 命令行安装mentohust, $ sudo dpkg -i mentohust_0.3.4-1_amd64.deb
    3. 配置IP,DNS
    4. 命令行设置mentohust, $ sudo mentohust -b1, 输入用户名和密码

三. 更新系统

  1. sudo apt-get update #更新源
  2. sudo apt-get dist-upgrade #更新系统
  3. sudo apt-get upgrade #更新软件

四. 命令安装小型软件

  1. # rar格式文件生成与解压
  2. sudo apt-get install rar unrar
  3. # makefile文件的自动生成程序
  4. sudo apt-get install automake
  5. # c,c++编译器和调试器
  6. sudo apt-get install gcc g++ gdb
  7. # flash插件
  8. sudo apt-get install flashplugin-installer
  9. # qtcreator是一个类似于VC++图形编程环境
  10. sudo apt-get install qtcreator

五. 中文输入法sogoupinyin

  1. 打开网页http://pinyin.sogou.com/linux/?r=pinyin
  2. 下载并安装sogoupinyin_2.0.0.0072_amd64.deb
  3. 在Language Support中把ibus改为fcitx
  4. 重启电脑
  5. 在soguo输入法配置，在Text entry中点击+添加sogoupinyin

六. 解决gedit中文乱码

由于windows下txt文件的默认编码方式为GBK, 而ubuntu默认不对GBK支持, 所以需要人为设置.

采用dconf-editor修改gedit的配置

  1. sudo apt-get install dconf-editor
  2. dconf-editor

在展开的org/gnome/gedit/preferences/encodings中
value一栏填入['UTF-8','GB18030','GB2312','GBK','BIG5','CURRENT','UTF-16']

安装CGAL

首先安装依赖库:

  • CMake (>= 2.8.11) 
  1. sudo apt-get install cmake-qt-gui cmake
  • Boost (>= 1.48)
    下载Boost, 进入解压目录:
  1. ./bootstrap.sh
  2. ./b2 --prefix=/usr/local/ #安装目录是/usr/local/
  3. sudo ./b2 install

  • Exact Arithmetic

    • GMP (>= 4.1.4)
      http://gmplib.org/
      or precompiled version that can be downloaded with CGAL-x.y-Setup.exe
      based on http://fp.gladman.plus.com/computing/gmp4win.htm

    • MPFR (>= 2.2.1)
      http://www.mpfr.org/
      or precompiled version that can be downloaded with CGAL-x.y-Setup.exe
      based on http://fp.gladman.plus.com/computing/gmp4win.htm

    • LEDA (>= 6.2)
      http://www.algorithmic-solutions.com/leda/index.htm

  1. sudo apt-get install libgmp-dev libgmpxx4ldbl
  2. sudo apt-get install libmpfr-dev libmpfr-doc
  3. sudo apt-get install libleda-dev

  • Visualization

    • Qt5 (>= 5.3)
      http://qt-project.org/

    • libQGLViewer
      http://www.libqglviewer.com/

    • Geomview
      http://www.geomview.org/
      Not supported with Visual C++

命令行安装 

  1. sudo apt-get install qt5-default qtcreator qt4-qtconfig qt4-demos qt4-designer
  2. sudo apt-get install libqglviewer-dev geomview
  • Numerical Libraries
    • EIGEN (>=3.1)
      Required by the packages:
      • Estimation of Local Differential Properties of Point-Sampled Surfaces
      • Approximation of Ridges and Umbilics on Triangulated Surface Meshes
      • Planar Parameterization of Triangulated Surface Meshes
      • Surface Reconstruction from Point Sets
        http://eigen.tuxfamily.org

下载并行矩阵计算软件包Eigen, 进入解压目录:

  1. mkdir build
  2. cd build
  3. cmake ../
  4. sudo make install
  - BLAS, LAPACK, ATLAS
   Required by the packages (if EIGEN is not available):
   * Estimation of Local Differential Properties of Point-Sampled Surfaces
   * Approximation of Ridges and Umbilics on Triangulated Surface Meshes
   * Planar Parameterization of Triangulated Surface Meshes
   http://www.netlib.org/blas/, http://www.netlib.org/lapack/
   or precompiled version that can be downloaded with CGAL-x.y-Setup.exe

  - MPFI
   Required by the package:
   o Algebraic Kernel
   https://gforge.inria.fr/projects/mpfi/
   (or shipped with RS http://vegas.loria.fr/rs/)

命令行安装

  1. sudo apt-get install libblas-dev liblapack-dev libatlas-dev
  2. sudo apt-get install libmpfi-dev
